# GUI Mode

Launch the native graphical interface:

```bash
abt gui
```

## 3-Step Wizard

1. **Select Source** — browse for an image file, enter a URL, or drag-and-drop
2. **Select Device** — choose from enumerated removable devices
3. **Write** — confirm and monitor progress with speed/ETA

## Theme System

6 built-in presets accessible from **View > Theme**:

| Theme     | Style                       |
| --------- | --------------------------- |
| Dark      | Default dark theme          |
| Light     | Light background            |
| Nord      | Arctic, pastel palette      |
| Solarized | Ethan Schoonover's palette  |
| Dracula   | Dark purple theme           |
| Monokai   | Warm syntax-inspired colors |

## Drag-and-Drop

Drop image files (`.iso`, `.img`, `.raw`, `.gz`, `.xz`, `.bz2`, `.zst`, `.qcow2`, `.vhd`, `.vhdx`, `.vmdk`) directly onto the window.

## Accessibility

- High-contrast palette (WCAG 2.1 AA compliant)
- Keyboard-only navigation
- Screen reader announcements for all state changes
- Reduced motion mode
- Large text mode
